# Multiplayer Rock Paper Scissors

A two-player realtime Rock Paper Scissors web game built with React, Vite, TypeScript, Node.js, Express, and Socket.IO.

## Run locally

```bash
npm.cmd install
npm.cmd run dev
```

- Client: `http://localhost:5173`
- Server: `http://localhost:3001`

## Verify

```bash
npm.cmd run test
npm.cmd run build
```
